# WarehouseApp
Web-based warehouse management system, C#, VisualStudio, MySQL Workbench

Configuration (appsettings.json)
The application uses `appsettings.json` for database connection configuration.

Example configuration:
{
  "ConnectionStrings": {
    "DefaultConnection": "Server=YOUR_SERVER;Database=WarehouseApp;Trusted_Connection=True;"
  }
